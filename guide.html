PlainList Guide

极简时间追踪与习惯打卡仪表盘的使用指引。

Terminal Commands

登录界面使用终端风格交互，以下是所有可用指令：

pl cd [name]登录到已有账号
pl new [name]创建新账号
pl onboard引导式新用户注册
pl ls列出所有账号
pl who显示当前登录用户
pl date显示当前日期与时间进度
pl graphic切换到图形化登录界面
clear清空终端屏幕
/help显示帮助信息

Quick Start

在终端输入 pl onboard 开始引导注册，或使用 pl new yourname 直接创建账号
设置口令（至少 3 位），完成后自动进入仪表盘
在 Day 板块底部点击 + Add habit or task 添加你的第一个习惯或任务
每天打开页面，勾选已完成的项目，追踪你的进度

也可以使用预置的演示账号体验：账号 admin，口令 admin

Dashboard Sections

01Now — 实时时钟 + 年/月/日进度条，一眼掌握时间消耗

02Day — 今日任务与习惯清单，点击勾选完成状态，底部显示完成率统计

03Month — 全月打卡表格视图，横向滚动查看每日完成情况，支持按周分组

04Year — 12 个月日历热力图 + 习惯一致性条带，全年趋势一目了然

05Week — 本周回顾：7 天卡片、雷达图、柱状图、四周趋势折线图

Tips

习惯（Habit）每天重复出现，任务（Todo）为一次性事项。

Day 与 Month 视图双向联动 — 在任一视图切换完成状态，另一视图立即同步。

点击导航栏右侧 lock 按钮可锁屏并返回登录界面。

同一标签页刷新后自动保持登录；关闭浏览器或新开标签页需重新登录。
